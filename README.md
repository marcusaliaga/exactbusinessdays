# ExactBusinessDays.com

Business days, working days, and deadline calculator by country, state, and province.

## Current MVP

This version includes:

- React/Vite front end
- Business-day counting excluding weekends
- Add business days
- Subtract business days
- Business days left this year
- About, Privacy, Terms, and Contact page structure
- Canonical homepage meta
- Basic structured data
- Sitemap and robots.txt

## Coming next

- Country selector
- State and province selector
- Public holiday lookup
- Included holiday list in the result
- Region-specific landing pages
